My Work

Find the details about some of my work deployed on GitHub below. Clicking on the images will take you to the respective deployed pages and clicking on the title will take you to the GitHub Repos.

PassWord Generator App

This is a PassWord Generator. It generates password based on the user input of character choices and lenght. The webpage is built using HTML, CSS and Vanila JavaScript.

Weather Dashboard App

The weather dashboard webpage helps you to search for current weather and also 5 day forecast of a city by using open weather map API. The webpage is built using HTML, CSS, Vanila JavaScript and APIs.

Work Day Scheduler App

The webpage displays a work day scheduler for Current/today's date. You can schedule events for each time blocks for the day and the time block starts from 9am and goes till 5pm. The time blocks are color-coded to indicate whether they are in the past, present, or future. To enter or edit an event, click into a time block. To save the event click on the save button for that time block. The events are cleared from local storage the next day when you load the webpage.

Word Search App

The webpage uses APIs to fetch defintion, references and examples for a searched word. You can add the word to favourties and storage it local storage. It also has a fun word Quiz challange as well as webpage translation using google translate API.
